# Sidcup Family Golf Website Clone

This project is a frontend clone of the Sidcup Family Golf website built to practice modern UI design and animations.

## 🚀 Technologies Used
- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)
- ScrollTrigger

## ✨ Features
- Custom animated cursor
- Smooth scroll-based animations
- 3D hover card effects
- Infinite scrolling text section
- Interactive UI elements

## 🎯 Purpose
The goal of this project was to improve my frontend development skills and learn how to create smooth animations and interactive user interfaces using GSAP.
